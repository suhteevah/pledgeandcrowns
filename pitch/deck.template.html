The idea in one breath

“ Pledge & Crown is a pixel-art RPG where the only way to act on the world is to write real, compilable Rust. The 10-act campaign covers a complete, university-level Rust curriculum — variables to async to unsafe to FFI. The Borrow Checker is an NPC who guards a bridge and rejects your code with the exact errors rustc would produce. By the time a player finishes the campaign, they are a junior-to-mid Rust developer.

The problem

Rust has the steepest learning cliff
of any mainstream language.

Books & tutorials

The official Book, Programming Rust, Rustlings. High quality — low engagement. Most readers abandon mid-ownership chapter.

Programming games

CodeCombat, Mimo, Zachtronics. High engagement — but none teach Rust. They do JS/Python/Lua or fictional assembly.

There is no third option.  We are the third option.

Why this exists03

The solution

You don’t read about Rust. You wield it.

Every meaningful action in the world is a function you write. The engine compiles it in a sandbox, runs it, and the world responds.

Combat is code Crafting is code Town-building is code Dialogue puzzles are code

Clean, clippy-idiomatic code hits harder. .unwrap() in the wrong place gets you hurt. The game has opinions — the same ones a senior reviewer would.

The Borrow Checker A robed figure of glowing reference symbols who guards a bridge. He speaks in real rustc errors. You cannot brute-force past him.

Technology

It eats its own dog food.

A Rust-teaching game, built entirely in Rust — the codebase is a teaching artifact.

Engine

Bevy ECS, native + WebGL2. bevy_egui + egui_code_editor drive the in-game IDE with real Rust syntax highlighting.

Player code

Source → server-side cargo compile → WASM. Runs in a fuel-limited sandbox — never the host process. The server owns the manifest.

Everywhere

One Rust codebase ships to desktop (native), browser (wasm), and mobile (Tauri 2.0, Android bundle builds today).
